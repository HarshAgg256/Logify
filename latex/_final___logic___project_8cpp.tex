{\itshape char temp} \+: Temporary variable used in swapping the elements of string during reversal.\hypertarget{_final___logic___project_8cpp_autotoc_md5}{}\doxyparagraph{Working}\label{_final___logic___project_8cpp_autotoc_md5}

\begin{DoxyEnumerate}
\item The input string is reversed first of all.
\item Then string is iterated to check if the character is an operand or an operator.
\item If an operand is found, then add it into the output string
\item If an operator is found, push it onto the stack if stack is empty. if the stack is filled already, then add the operators at the top of stack to the ouput string till the stage where precedence of current operator becomes greater than that of the operator on top of stack.
\item If a closing parenthesis is found, simply push it in the stack. if an opening parenthesis is encountered, then add the operators at the top of stack to the string till the stage where it encounters closing parenthesis in the stack.
\item Now empty the stack and add the operators to the ouput string.
\item Finally, Reverse the output string to generate prefix notation.

This function find the truth value of the logical expression. 

The function iteratively finds the three consecutive characters from the string array in postorder form so that the first two characters are operands and the last one is an operator. It then simplifies the three characters too give either 0 and 1. The process then repeats till one final value is obtained.
